deltacoverage:deltacoverage

Full name:

org.apache.maven.plugins:maven-deltacoverage-plugin:1.0-SNAPSHOT:deltacoverage

Description:

Mojo to generate delta coverage report. Entry point for maven plugin. Delta coverage is calculated by taking into consideration current code coverage and code changed since a specific baseline version. Hence needs parameters coverageReport and baselineVersion.

Attributes:

  • Requires a Maven 2.0 project to be executed.

Optional Parameters

Name Type Since Description
baselineVersion String - CVS branch/tag name to be used as baseline for delta calculation.
coverageReport File - Path of coverage report xml.
Default value is: ${project.reporting.outputDirectory}/cobertura/coverage.xml.
reportDir File - Location where output report should be generated.
Default value is: ${project.build.directory}.
scm String - The SCM repository to use. Currently supported - CVS (default).
Default value is: CVS.
sourceDir File - Location where source code resides
Default value is: ${project.build.sourceDirectory}.

Parameter Details

baselineVersion:

CVS branch/tag name to be used as baseline for delta calculation.
  • Type: java.lang.String
  • Required: No
  • Expression: ${delta.baseline}

coverageReport:

Path of coverage report xml.
  • Type: java.io.File
  • Required: No
  • Expression: ${delta.coverage.reportPath}
  • Default: ${project.reporting.outputDirectory}/cobertura/coverage.xml

reportDir:

Location where output report should be generated.
  • Type: java.io.File
  • Required: No
  • Expression: ${delta.reportDir}
  • Default: ${project.build.directory}

scm:

The SCM repository to use. Currently supported - CVS (default).
  • Type: java.lang.String
  • Required: No
  • Expression: ${delta.scm}
  • Default: CVS

sourceDir:

Location where source code resides
  • Type: java.io.File
  • Required: No
  • Default: ${project.build.sourceDirectory}
